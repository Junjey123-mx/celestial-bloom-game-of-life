# Celestial Bloom

Simulación de Conway's Game of Life desarrollada en Rust con raylib y un framebuffer propio.

El proyecto mostrará una flor estable construida con still lifes, rodeada por oscillators y spaceships sobre un fondo blanco con células vivas de color celeste.

## Estado

Proyecto inicializado. La lógica del Game of Life y el render loop se implementarán progresivamente.
